How to install and Use VS code's Live Server Extension

Install | Use | Video

Installation Instructions

  1. In Visiual Studio Code, click the Extensions icon in the left column toolbar.
  2. In the seach bar that appears, type Live Server.
    3. Select the Live Server extension. It should be top choice.
  3. In the page for Live Server that appears, click the green Install button.

Usage Instructions

  • Visual Studio Code must must be open to a folder to use Live Server.
  • In VS Codes's Explorer, right click (windows) or Control+click (macos) on an HTML file. From the context menu that appears, select Open with Live Server.
